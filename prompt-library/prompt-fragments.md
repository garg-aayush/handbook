# Prompt Fragments

Reusable prefixes, suffixes, and anecdotes to mix into other prompts.

## Suffixes

### Clarify Intent Before Executing

Use when starting a long-running agent task (e.g., Claude Code, Codex) to ensure the agent understands your actual goal before planning.

```
Interview me until you have 95% confidence about what I actually want, not what I think I should want.
```

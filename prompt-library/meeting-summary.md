## Version 2 — Structured Summary

```
Please provide a comprehensive yet concise summary of this meeting in markdown format.

<transcript>
{{TRANSCRIPT}}
</transcript>

<output_format>
- Key Discussion Points: Outline the main topics covered and any significant insights shared
- Action Items: List all tasks with clear owners, deadlines, and success criteria
- Decisions Made: Document all decisions with their rationale and implementation timeline
- Questions for Follow-up: Note any unresolved questions that require additional research or discussion
- Next Meeting Focus: Suggest priority topics for the next discussion
</output_format>

Format the summary in a structured, scannable layout with bullet points for quick reference. Highlight urgent items that require immediate attention.
```
